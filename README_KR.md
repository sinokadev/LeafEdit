# LeafEdit: 모두를 위한 텍스트 에디터!

> **참고: 이 소프트웨어는 개발중이므로, 소개된 모든 기능이 지원되지 않습니다.**
> **참고: 한국어 버전, 번역되지 않은 내용이 있을 수 있습니다.**

즉시 모드 GUI를 사용하는 텍스트 에디터입니다. 간단하고 가볍습니다. 파일 트리와 바이너리 헥스 에디팅을 지원합니다. LeafEdit은 오픈소스이며, 무언가 아쉽게 느껴지면 편하게 PR 날려주세요!

아직까지 Windows와 macOS는 지원 고려사항에 없으며, 최신 버전의 Ubuntu와 Fedora를 우선 지원합니다.
